User Stories

1. As a patient, I want to see my doctor's availability online so that I can book an appointment quickly.
2. As a patient, I want to see keep track of my appointments so that I can easily retrive my medical history.
3. As a patient, I want to see which doctor's are taking new patients so that I don't have to waste time calling all doctors in my area.
4. As a doctor, I want to allow my patients to book online so that I can reduce my receptionist's workload.
5. As a doctor, I want to keep track of my appointment time lengths so that I can identify how long certain types of appointments take.
6. As a doctor, I want to input appointment notes to that i can easily retrieve a patient's medical history.
7. As a doctor, I want to keep track of patient no-shows so that I can identify potential problem areas for scheduling.
